What's new in this drop

  • Added rewriting support to DOM AST. There are two set of API to perform rewriting.
    • An AST-modifying rewrite API
      To use the AST-modifying rewrite the client must create an AST from an existing compilation unit (ASTParser#createAST(...)). Then the client need to start the record process of the AST changes (CompilationUnit#recordModifications()) and modify the AST after. When modifications are done, the client can call a method to compute a text edit tree containing all the text changes (CompilationUnit#rewrite()).

When modifications are done, the client can call a method to compute a text edit tree containing all the text changes (NewASTRewrite#rewrite()).

 *

 * such a conflict:

 * 	the underlying resource even though this working copy was created

 * the corresponding compilation unit.

 *

 * - The first call to getWorkingCopy(WorkingCopyOwner, IProblemRequestor, IProgressMonitor)
